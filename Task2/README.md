# Car Price Prediction using Machine Learning

This project predicts the selling price of cars using machine learning.

## Objective
To develop a regression model that estimates the selling price of a car based on features such as fuel type, kilometers driven, seller type, transmission type, and showroom price.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Machine Learning Model
Random Forest Regressor

## Model Performance
R² Score: 0.96

## Key Insights
The most important feature influencing the selling price is the present showroom price of the car, followed by car age and kilometers driven.
